# ReactNative-Network-App
为了方便一些外贸、技术人查阅国外的文档，及访问一些被误伤限制的正规网站，因此就诞生了这个软件，功能主要分客户端+服务端两个部分，客户端主要是APP，服务端就是基于S协议的代理软件。
